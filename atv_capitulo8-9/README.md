## Descrição
O layout busca um equilíbrio entre a modernidade e a sobriedade, utilizando os seguintes pilares visuais:

Identidade Cromática de Alto Contraste: O uso do cinza-escuro (grafite) como base principal confere um tom robusto, técnico e profissional ao site. O laranja atua como a cor de destaque (foco), direcionando os olhos do usuário imediatamente para a barra lateral de navegação e para os títulos principais, simulando o comportamento da imagem de referência.

Construção Modular e Limpa: Em vez de elementos flutuantes ou desalinhados, o layout é dividido em "blocos" de conteúdo bem delimitados (com bordas sutis e variações de tons de cinza). Isso melhora a leitura e dá um aspecto organizado e geométrico.

Tipografia Focada na Leitura: A escolha por fontes sem serifa modernas (como Segoe UI ou Helvetica) garante que o texto permaneça legível tanto em telas de computadores quanto em celulares.

Fluidez e Adaptação: O design não é rígido. Graças ao Flexbox, os blocos se movem de forma inteligente: na tela grande, o menu se posiciona como uma coluna lateral elegante; nas telas menores, ele se move para baixo, empilhando-se de forma natural para não quebrar a experiência do usuário.
