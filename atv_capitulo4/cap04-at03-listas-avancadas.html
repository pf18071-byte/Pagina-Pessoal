2. Ranking de Tecnologias

  1. Frameworks de CSS:

    Eles são aceleradores. Permitem criar interfaces bonitas e profissionais em tempo recorde, sem que você precise escrever cada linha de CSS do zero.

  2. Frameworks/Libraries JS:

    Hoje, quase toda aplicação moderna usa um desses. Eles gerenciam o estado da página e tornam o desenvolvimento de sistemas complexos muito mais organizado.

  3. JavaScript (JS):

    É o "músculo" e o "cérebro". É a única linguagem de programação que roda nativamente em todos os navegadores. Sem ele, não teríamos interatividade, animações complexas, nem carregamento de dados em tempo real (como o feed das redes sociais).

  4. CSS (Cascading Style Sheets):

    É o que dá forma, cor, hierarquia visual e acessibilidade estética. Sem o CSS, a internet seria uma lista infinita de textos pretos em fundo branco com links azuis.

  5. (HyperText Markup Language):

    O esqueleto. Sem HTML, não existe web. É a linguagem que dá significado à informação e permite que o navegador saiba o que é um título, um parágrafo ou um link.

3. Glossário Técnico

HTML:
Linguagem de marcação utilizada na construção de páginas web.
Sigla para HyperText Markup Language.
URL:
Endereço Web:
Caminho padrão utilizado para localizar e acessar recursos na internet.
JavaScript:
Linguagem de programação de alto nível que permite implementar itens complexos em páginas web interativas.
HTTP:
HTTPS:
Protocolos de transferência de dados que permitem a comunicação entre cliente e servidor na web.
DOM:
Document Object Model.
Interface de programação que estrutura documentos HTML como uma árvore de objetos para serem manipulados por linguagens como JavaScript.
CSS:
Linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML.
API:
Interface de Programação de Aplicações que permite que diferentes softwares se comuniquem.
DNS:
Sistema que traduz nomes de domínios legíveis por humanos em endereços IP.
